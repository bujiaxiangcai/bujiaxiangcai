是准备好下次和哥吃饭要送的礼物，因为哥一直拒绝我出门的邀请，太生气了就把项链放进口袋想着随便给个人得了。不过幸好，它的主人重新接受了他。

虽然哥喜欢随意活着，但告白这种事还是不要太随意了。

所以我说：“除了做爱，能不能也试试和我交往？”
